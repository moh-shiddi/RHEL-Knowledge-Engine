معجم أوامر RHEL العربي — الإصدار V2

الملفات:
- index.html
- style.css
- app.js
- commands.json

المميزات:
- فهرس بحث عربي سريع.
- فهم كلمات مترادفة مثل تنزيل/تثبيت وبرنامج/حزمة.
- اقتراحات تلقائية أثناء الكتابة.
- تظليل الكلمات المطابقة.
- المفضلة وسجل البحث عبر localStorage.
- وضع ليلي ونهاري.
- عرض شبكي أو قائمة.
- فلاتر التصنيف والخطورة وsudo وإصدار RHEL.
- تفاصيل موسعة ونسخ الأمر أو المثال.
- دون مكتبات خارجية.

التشغيل محلياً:
1. افتح موجه الأوامر داخل المجلد.
2. نفّذ:
   python -m http.server 8000
3. افتح:
   http://localhost:8000

يمكن رفع الملفات كما هي على Cloudflare Pages أو GitHub Pages.

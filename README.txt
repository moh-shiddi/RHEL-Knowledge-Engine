محرك معرفة RHEL العربي — المرحلة الثالثة (Intent Engine V1)

الملفات الجديدة:
- intents.json: تعريف نوايا البحث والكلمات والوجهات والمتغيرات.
- intent-engine.js: تحليل الجمل، التصحيح التقني، استخراج المتغيرات، وترتيب النتائج.
- intent-tests.json: مجموعة اختبارات لعبارات عربية وفصحى ولهجية.
- test-intents.js: تشغيل الاختبارات بواسطة Node.js.

مميزات المرحلة الثالثة:
- فهم عبارات مثل: "السيرفس failed" و"افتح بورت 8080" و"أبي أثبت ngnix".
- تصحيح أخطاء تقنية شائعة مثل ngnix → nginx وsystmctl → systemctl.
- استخراج PACKAGE وSERVICE وPORT وPROTOCOL وHOST وUSER وPATH وغيرها.
- تعبئة متغيرات Workflow تلقائياً عند فتح المسار المقترح.
- رفع نتيجة المهمة أو حل المشكلة المناسب إلى أول النتائج.
- عرض بدائل عندما تكون العبارة محتملة لأكثر من نية.
- يعمل محلياً بالكامل ودون API أو إرسال بيانات المستخدم.

الرفع إلى GitHub:
ارفع جميع الملفات واستبدل الملفات الموجودة. لا تغيّر إعدادات Cloudflare Pages.

اختبار المحرك محلياً:
node test-intents.js

تشغيل الموقع محلياً:
python -m http.server 8000
ثم افتح http://localhost:8000

تطوير النوايا:
عدّل intents.json ثم أضف حالات إلى intent-tests.json وشغّل الاختبار قبل الرفع.

# RHEL Arabic Knowledge Engine — Phase 5

## Guided Workflow Runner

المرحلة الخامسة تحول كل مهمة أو حل مشكلة إلى جلسة موجهة:

1. مراجعة المتطلبات.
2. تعبئة المتغيرات مثل `<SERVICE>` و`<PACKAGE>`.
3. تنفيذ خطوة واحدة في كل مرة.
4. تسجيل نجاح الخطوة أو وجود مشكلة.
5. التحقق من النتيجة.
6. فتح الأخطاء الشائعة والفحوصات.
7. مراجعة خطة التراجع.
8. نسخ أو تنزيل تقرير الجلسة.

## المميزات

- يعمل على جميع المهام وحلول المشكلات التي تحتوي خطوات.
- يستنتج المتغيرات الناقصة تلقائياً من نص الأوامر.
- يحفظ الجلسات في المتصفح ويستكمل آخر جلسة.
- لا ينفذ أي أمر تلقائياً.
- يحافظ على Linux Doctor ومحرك النية والبحث الحالي.

## الاختبار

```bash
node --check app.js
node --check workflow-engine.js
node test-workflows.js
```
